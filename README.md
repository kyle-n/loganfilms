# Logan Films

A simple local webserver that polls movie theaters in the Logan, Utah, area to check what movies are playing.

## Running locally

- [Install Go](https://go.dev/doc/install)
- [Install Air](https://github.com/air-verse/air)
- Clone this repo
- Open the repo from the command line and run Air
- Open [http://localhost:8080/](http://localhost:8080/) in a browser
